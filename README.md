# Goal
维护一个可以随时一键部署同步开发环境的代码库
                                                                                                      
# 特点                                                                                                                                    
1. 快速备份还原.vimrc .bashrc等配置                                                          
2. 提供单例模式等装饰器、redis/elasticsearch等db库的python封装供更简便的调用                 
3. 通过pip -r requirements一键安装项目中所需库到当前项目路径下的libs目录，避免权限等问题  
4. 存放一些脚本、测试代码
